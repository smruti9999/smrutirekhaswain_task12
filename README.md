Formula For Calculating compound interest is 
A = P x (1+r/n)^nt where
A = Amount
P = Principal
r = rate of interest
N = number of times interest is compounded per year
T = Time(in years)
The Compound interest is calculated as = A - P

Display the result using console.log()

How to Run this file
Open the file in web browser
Then go to developer tools
Select the console tab
View the output.
